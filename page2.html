The project that I chose was the earthquake map, I chose this becuse it was the first website for App Academy that showed a world-wide map of the different magnitude of earthquakes. I chose this project because it was interesting to learn about the different types of earthquakes in the world and places numerous map markers of earthquakes around the map. To create this project I used the API for Google Maps to display the map of the world and the magnitude of each earthquake. This project works by the user clicking the map markers on the page to show the magnitude and the location of the earthquake.
